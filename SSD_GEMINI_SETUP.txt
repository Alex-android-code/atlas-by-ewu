ATLAS Gemini setup on SSD

1. Open D:\ATLAS_EWU
2. Copy .env.example to .env
3. Open .env and replace:
   GEMINI_API_KEY=your_gemini_api_key
   with your real Gemini API key.
4. Keep:
   ATLAS_AI_PROVIDER=gemini
   GEMINI_MODEL=gemini-2.5-flash
5. Start ATLAS from the SSD folder.

Do not send this .env file to anyone. It contains the private API key.
