# EWU European Welders Union 4.0 Production

## Start
1. Распакуй архив в новую папку.
2. Скопируй `.env.example` в `.env`.
3. Вставь Telegram/Gemini/чат-ID.
4. Установи:
`pip install -r requirements.txt`
5. Запусти:
`python bot.py`
