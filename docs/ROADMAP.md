# ROADMAP

Each stage must be implemented separately. Do not mix multiple stages in one uncontrolled implementation.

## Stage 01: Project Foundation And Product Constitution

Define ATLAS identity, documentation, founder rights, roles, languages and core product rules.

## Stage 02: Language Auto-Detection And I18n Foundation

Status: foundation prepared.

Separate UI language, conversation language, URL language, browser language and saved user preference.

## Stage 03: Human One-Question Chat Behavior

Status: foundation prepared.

Make ATLAS ask like a human, with maximum one question per message.

## Stage 04: Human Map / Candidate CV Map

Gradually build a professional candidate profile from conversation.

## Stage 05: Employer Map And Job Offer Map

Gradually build structured employer and vacancy profiles from conversation.

## Stage 06: Admin Panel Foundation

Prepare controlled admin operations, review queues and basic approval flows.

## Stage 07: Skill-Based Matching Engine

Match people and opportunities by skills, experience, documents, readiness and goals.

## Stage 08: Payment Gate And Anonymized Candidate Preview

Protect candidate data and unlock access only after payment or admin approval.

## Stage 09: Courses Module

Connect training and skill development services.

## Stage 10: Legalization Module

Support document, visa, permit and legalization workflows.

## Stage 11: Insurance Module

Support insurance service discovery and partner connection.

## Stage 12: Housing Module

Support housing options and relocation needs.

## Stage 13: Partner Map And Service Map

Map trusted partners and services across Europe.

## Stage 14: PDF CV Export

Export candidate CV maps into professional PDF documents.

## Stage 15: Security, Roles And Audit Logs

Harden access control, audit important actions and protect sensitive data.
