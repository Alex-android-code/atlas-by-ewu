# PROJECT CONSTITUTION

## Identity

Project name: **ATLAS by EWU**

Ecosystem owner: **EWU**

Founder: **Oleksandr Didkivskyi / Aleksandr Didkowski**

## Mission

ATLAS by EWU helps people in Europe build a stable work-life path through work, skills, documents, training, insurance, housing and trusted partners.

## Product Nature

ATLAS is an AI-powered workforce and life-support ecosystem for Europe.

It is not only a job board.
It is not only a chatbot.
It is not a simple recruitment agency.
It is not limited to welders or any other single profession.

ATLAS must support all professions, skills, industries and career paths.

## Core Formula

Human conversation
-> Human Map / Candidate CV Map
-> Employer Map / Job Offer Map
-> Service Map
-> Matching / Life Path Engine
-> Payment or admin approval
-> Connection through EWU

## Core Maps

1. Human Map / Candidate CV Map
2. Employer Map
3. Job Offer Map
4. Service Map
5. Partner Map
6. Match Map
7. Life Path Map

## Core Rule

ATLAS asks like a human and thinks like a system.

## Conversation Principle

One ATLAS message must contain maximum one question.

## User Privacy Principle

Employers must not see candidate personal data before payment or admin approval.

Candidate contacts, photos, documents and private notes are protected by default.

## Scalability Principle

ATLAS must support all professions and skills, not a closed profession list.

Countries, languages, professions, skills and services must be extendable through configuration or admin-managed taxonomy.
